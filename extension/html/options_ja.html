一般

インストール

以下のツールをインストールしてください。

  1. TortoiseSVN をダウンロードページからダウンロードし、インストールしてください。
    既にインストール済みなら、この手順をスキップしてください。
  2. install.exe をダウンロードしてください。
    install.exe を実行し、インストールしてください。
    このツールは Chrome から TortoiseSVN を開くためのツールです。

アンインストール

  1. uninstall.exe をダウンロードしてください。
    uninstall.exe を実行し、アンインストールしてください。

TortoiseProc.exe のパス

TortoiseProc.exe のパスの設定をしてください。

Subversion リポジトリの URL

Subversion リポジトリの URL を追加してください。

登録した URL をクリックすると、 TortoiseSVN が開かれます。
例) "http://svn_repos/" を追加し、 "http://svn_repos/dir" のリンクをクリックすると、 TortoiseSVN で開かれます。

    Open TortoiseSVN について

    概要

    この拡張機能では TortoiseSVN をブラウザから直接開くことができます。
    登録済みの URL へのリンクをクリックすると、 TortoiseSVN のリポジトリブラウザが開かれます。

    この拡張機能はコンテキストメニューに TortoiseSVN リポジトリブラウザ、ログ、注釈履歴を開くメニューを追加します。

    開発者向け情報

    HTML のソースを編集し、 data-tsvn-info の属性を "a" タグに追加すると、 TortoiseSVN の既定の動作を指定できます。

    HTML 4.1 の rel 属性を data-tsvn-info 属性の代わりに使うこともできます。

    • TortoiseSVN ログを開く場合： 
      <a href="http://svn_repos/" data-tsvn-info="tsvn[log]">Open Log Viewer</a>
    • TortoiseSVN ログの特定のリビジョンを開く場合： 
      <a href="http://svn_repos/" data-tsvn-info="tsvn[log][12,8]">Open Log Viewer</a>
    • TortoiseSVN 注釈履歴を開く場合： 
      <a href="http://svn_repos/" data-tsvn-info="tsvn[blame]">Open Blame Viewer</a>
